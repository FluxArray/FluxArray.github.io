~ > Philosophy

These are the principles that guide my work, inspired by UNIX philosophy and modern UX practices.

Simplicity

Complexity is the enemy of reliability. I strive to write code that is easy to read, easy to delete, and does one thing well.

Robustness

Systems should fail gracefully. Whether it's handling API errors or user input, I design for the "happy path" but engineer for the edge cases.
